# git 常用操作命令

### 开始一个工作区 (参见	: git help tutorial)
|命令|作用|
|-----|-----|
|clone|克隆一个仓库到一个新目录|
|init|创建一个控Git	仓库或从新初始化一个已存在的仓库|

### 在当前变更上工作	(参见	: git help everyday)
|命令|作用|
|-----|-----|
|add|添加文件内容至索引|
|mv|移动或重命名一个文件、目录或符号链接|
|reset|重置当前HEAD	到指定状态|
|rm|从工作区和索引中删除文件|

### 查看历史和状态	(参见	: git help)
|命令|作用|
|----|----|
|bisect|通过二分查找定位引入	bug	的提交|
